Run with
`node roshambo.js --move=rock|paper|scissors `
optional flags:
- `--name=<your name>`
- `--type=Normal|LizardSpock`

example:
```
node roshambo.js --move=spock --name=Anthony --type=LizardSpock
Playing a game of LizardSpock against the computer.
Anthony plays spock!
Computer plays scissors!
~Anthony wins.~
```
